\doxysubsection{Detailed Description}
Circular Equatorial Data Reader Class. 

\mbox{\hyperlink{class_c_e_d_r}{C\+E\+DR}} stands for circular, equatorial data reader. This class collects methods which read the output of a data run (i.\+e., the H\+D\+F5 data produced by the code) and present it in human readable format.
