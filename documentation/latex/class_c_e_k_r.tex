\doxysubsection{Detailed Description}
Circular Equatorial \mbox{\hyperlink{class_kerr}{Kerr}} Radiation Class. 

\mbox{\hyperlink{class_c_e_k_r}{C\+E\+KR}} describes methods for computing various quantities to radiation from circular, equatorial \mbox{\hyperlink{class_kerr}{Kerr}} geodesics. It relies on input from instances of the classes \mbox{\hyperlink{class_s_w_s_h}{S\+W\+SH}}, \mbox{\hyperlink{class_f_t}{FT}}, and \mbox{\hyperlink{class_c_e_k_g}{C\+E\+KG}}. ~\newline
